# prime_video
This project focuses on analyzing the content available on Amazon Prime Video using data analytics tools. It provides insights into movies and TV shows based on different attributes such as genre, ratings, release year, and country.

# Features
. Total Movies & TV Shows Analysis
. Genre-wise Distribution
. Ratings Breakdown
. Country-wise Content Availability
. Release Year Trends
. Interactive Dashboard (Power BI)
# Tools & Technologies
. Power BI
